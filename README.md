### Installation

<h4 align="center">Server</h4>

##### Clone the repository

```sh
git clone https://github.com/Yoru-cyber/PasswordManager
```

##### Go into project and install dependencies

```sh
cd PasswordManager
```

```sh
composer install
```

##### Run local server

```sh
symfony server:start
```
